# NVoltenerergy SRL website

Static demo website for NVoltenerergy SRL.

## Files

- `index.html` - main page
- `assets/styles.css` - visual styling and responsive layout
- `assets/app.js` - language switcher, modal, slider, filters, and form demo logic
- `assets/` - logo, favicon, and placeholder project images

## Preview

Open `index.html` directly in a browser, or serve the folder with any static server.
